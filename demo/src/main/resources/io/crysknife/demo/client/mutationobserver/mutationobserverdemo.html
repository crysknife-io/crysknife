The MutationObserver interface provides the ability to watch for changes being made to the DOM tree. It is designed as a replacement for the older Mutation Events feature, which was part of the DOM3 Events specification. 
        @Inject\n
        protected HTMLButtonElement test;\n
            \n
        @Inject\n
        protected MutationObserver observer;\n
            \n
        @OnAttach("test")\n
        protected void onAttach(MutationRecord mutationRecord) {\n
            DomGlobal.console.log("on attach " + mutationRecord);\n
            textBox.value = "    on attach " + mutationRecord;\n
        }\n
        \n
        @OnDetach("test")\n
        protected void onDetach(MutationRecord mutationRecord) {\n
            DomGlobal.console.log("on detach " + mutationRecord);\n
            textBox.value = "    on detach " + mutationRecord;\n
        }\n
